# poster-mapの使い方
## 概要
* このシステムは安野たかひろさんチームが公開したオープンソースを元にカスタマイズしたものです
* Googleスプレッドシートに入れたデータをnetlifyで作ったサイトに自動デプロイするまでの手順を記します

## 準備するもの（環境）
* Google Spreadsheet
* Linuxサーバ
* Github アカウント
* netlify アカウント(githubアカウントでログインできます)

## 準備するもの（データ）
* 掲示板データのcsv
** area,name,lat,long,status,noteの形に編集したもの
